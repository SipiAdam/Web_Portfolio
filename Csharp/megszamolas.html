A megszámolásban egy adott feltétel alapján megszámoljuk a bemenet elemeit. És utána ezt a számot adjuk vissza. Ezt a 3000-es feladatokba sokat használtuk.


                static int Hány_páros_van(List<int> lista)
                {
                    int páros = 0;
                    for (int i = 0; i < lista.Count; i++)
                    {
                        if (lista[i] % 2 == 0)
                        {
                            páros++;
                        }
                    }
            
                    return páros;
                }
