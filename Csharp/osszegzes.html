Programozási tételek - Összegzés

Az összegzésnél végig megyük a bemenetünkön és összegzünk bizonyos elemeket vagy akár az egész bemenetet.


                static int Összegzés(int[] t)
                {
                    int result = 0;
                    for (int i = 0; i < t.Length; i++)
                    {
                        result += t[i];
                    }
                    return result;
                }
