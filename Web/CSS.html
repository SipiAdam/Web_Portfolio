CSS

CSS (Cascading Style Sheets) egy stílusleíró nyelv, amelyet weboldalak megjelenésének formázására használnak. A CSS lehetővé teszi a felhasználók számára, hogy elkülönítsék a dokumentum tartalmát (HTML) és annak megjelenését, így könnyen testreszabhatóvá válnak a weboldalak designjai. A CSS segítségével definiálhatók a szöveg formázása, a betűtípusok, a színek, a margók, a háttérképek és egyéb vizuális elemek.
