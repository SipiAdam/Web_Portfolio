JavaScript

JavaScript egy dinamikus, magas szintű programozási nyelv, melyet általában weboldalak interaktivitásának és dinamikusságának létrehozására használnak. A böngészőkben fut, lehetővé téve a felhasználói események kezelését, az adatok manipulálását, valamint a weboldalak dinamikus tartalmának generálását és frissítését. JavaScript széles körben elterjedt a webfejlesztésben, de használható asztali alkalmazásokban és szerveroldali fejlesztésben is (Node.js környezetben).

                
                    // Egyszerű JavaScript példa
                    function greet(name) {
                        return 'Hello, ' + name + '!';
                    }
            
                    // Függvény meghívása és eredmény kiírása a konzolra
                    let greeting = greet('World');
                    console.log(greeting);
